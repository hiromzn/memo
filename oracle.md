# proc

- usage

```
proc option=value...
```

- sample

```
proc myfile
proc INAME=myproc

```

- priority of option

  - オプションの値は、優先順位の低いものから順に、次のように決定される。

    - プリコンパイラに組み込まれている値
    - Pro*C/C++システム構成ファイル内の値の集合
    - Pro*C/C++ユーザー構成ファイル内の値の集合
    - コマンドラインで設定される値
    - インラインで設定される値
